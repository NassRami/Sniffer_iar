Information Center for RL78 | EXAMPLES

Example description

The startup-sample includes the following functions:
- Initialization of the on-chip-debug interface
- Basic initialization of the microcontroller
- Setup of a 10 ms interval timer
- Setup of INTP1 as external interrupt
- Setup of I/O port

At start LED01 is flashing in a 1 s interval.
Each click of switch SW1 increases the interval time of LED01 by 100 ms.

The sample project includes two different targets:
- Debug
- Release

The target ‘Debug’ is configured to start a debug session using E1, the target ‘Release’
is configured to generate a intel-hex file that can be used by an Renesas Flash Programmer
(e.g PG-FP5 or RFP).

Before starting the sample, please make sure that the connection to the target board is correct.

For details please refer to the following User's Manuals:
- E1/E20 Emulator User's Manual: R20UT0398EJ0300.
- E1/E20 Emulator Notes on Connection for RL78: R20UT1994EJ0200_E1E20_RL78.
