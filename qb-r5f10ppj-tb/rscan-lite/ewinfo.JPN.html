Information Center for RL78 | EXAMPLES

Demonstrates the RL78 RSCAN-Lite Macro

Instructions

Build and download the sample code to the targetboard by pressing the 'Build and Download' button on the debug tool bar. Click the 'Go' button to start program execution.
Three tests selectable by user in main function are included:

EE_RSCAN_COM_FIFO_TX_and_RX_FIFO_Test

• Test the TX/RX FIFO (COM FIFO ) for one Channel
• Test RX FIFO for one Channel
• Transmit Test Messages with the TX/RX FIFO
• Push Messages into the FIFO
• Check TX/RX FIFO Full Flag
• Receipt Test Messages with RX FIFO
• Create and Test Interrupts for Transmission and Reception

EE_RSCAN_TX_RX_MB_Test

• Test the TX Message Buffer for one Channel
• Test the RX Message Buffer for one Channel
• Create Interrupt for Transmission
• Transmit Test Message over TX Message Buffer
• Push Message into Buffer
• Wait till Transmission Interrupt Occur
• Receipt Messages in RX Message Buffer

EE_RSCAN_TX_RX_MB_Loop_Test

• Same Test as “EE_RSCAN_TX_RX_MB_Test”
• Communication over intern Loop Back
